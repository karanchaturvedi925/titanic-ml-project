# Titanic Survival Prediction 🚢

## Project Overview
Predicted survival of Titanic passengers using Machine Learning.
Achieved 81% accuracy using Logistic Regression.

## Dataset
- 891 passengers data
- Features: Age, Gender, Ticket Class, Fare, etc.

## Steps Followed
1. Data Loading
2. Data Cleaning (Missing values handled)
3. Feature Encoding
4. Model Training
5. Accuracy: 81%

## Libraries Used
- Pandas
- Scikit-learn

## Result
Model accurately predicts survival with 81% accuracy.
